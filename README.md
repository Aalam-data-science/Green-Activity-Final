# 🚀 GitHub Green Streak Automator

A professional Python-based automation tool designed to manage and visualize GitHub contribution activity. This project demonstrates expertise in **Python Scripting**, **Git Version Control**, and **Workflow Automation**.

## 🌟 Project Overview
This tool is built to automate the process of creating historical commits, allowing developers to maintain a consistent contribution graph. It uses Git's internal metadata handling to accurately reflect activity over a defined period.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Version Control:** Git
- **Environment:** Optimized for Windows PowerShell / Command Prompt

## 🚀 Key Features
- **Historical Commit Generation:** Ability to backdate commits to any specific year or month (e.g., Full 2025 activity).
- **Natural Activity Simulation:** Configurable commit frequency (2-4 commits per day) to mimic realistic coding patterns.
- **Automated Staging:** Handles `git add` and `git commit` processes automatically via Python's `os` module.
- **Metadata Management:** Directly manipulates `GIT_AUTHOR_DATE` and `GIT_COMMITTER_DATE` for precision.
